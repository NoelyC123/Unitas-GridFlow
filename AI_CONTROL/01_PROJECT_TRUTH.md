# Project Truth

## What Unitas GridFlow is

Unitas GridFlow is a narrow pre-CAD QA / compliance / submission-readiness tool
for UK electricity network survey-to-design handoffs.

It is a local Flask/Python web app that:
- accepts survey CSV data uploads
- normalises the input schema
- runs DNO-specific QA checks against rulepack rules
- generates issue outputs and map data
- renders results on a Leaflet map
- produces PDF QA reports
- retains job outputs locally for review and download

## What Unitas GridFlow is not

It is not:
- a broad software platform
- a CAD/GIS replacement
- a general utility SaaS
- a full engineering platform
- an AI-first design product
- a digital twin system
- a multi-tenant enterprise product

## Best realistic identity

- DNO survey compliance gatekeeper
- pre-CAD submission-readiness validator
- narrow workflow engine for survey-to-design QA

## Best commercial form

1. internal / consultancy leverage first
2. licensed niche product second
3. broader software ambitions only if repeatability and proof-of-value are proven

## Current strategic judgment

- worth continuing
- only in narrowed form
- only under proof-of-value logic
- do not broaden scope
